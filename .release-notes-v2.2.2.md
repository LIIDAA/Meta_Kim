# v2.2.2 — Close Prism v2.2.0 review HIGH findings

Targeted patch closing all 3 HIGH findings from the independent v2.2.0 quality review (`docs/v2.2.0-prism-review.md`). No production hook changes — `spine-state.mjs` and `enforce-agent-dispatch.mjs` remain frozen per Warden ruling.

## Fixed

### H1 — Rule-ID to Packet-Name vocabulary mapper
- `config/contracts/deliverable-type-profiles.json` ships an explicit `ruleToPacketMap` block (schemaVersion `1.0.0 → 1.1.0`) mapping the 7 core contract rule IDs to spine-state packet names:
  - `testStrategyDefined → testStrategyPacket`
  - `rollbackPlanDefined → rollbackPlanPacket`
  - `structureHygiene → structureHygienePacket`
  - `interfaceContract → interfaceContractPacket`
  - `sideEffectLedger → sideEffectLedgerPacket`
  - `permissionMatrix → permissionMatrixPacket`
  - `linkValidation → linkValidationPacket`
- `canonical/runtime-assets/shared/lib/policy-registry.mjs` exposes new exported helper `resolvePacketName(registry, ruleId)` with `fallback_to_rule_id` policy for forward compatibility.
- **Unblocks v2.3.0 R4** hook wiring (registry consumption).

### H2 — Q4 honesty in progress doc
- `progress-v2.2.0.md` now states explicitly that v2.2.0 ships only the **data layer** of Q4 (the `requiresConfirmation` field). The **enforcement layer** (a PreToolUse interceptor pausing execution before first file write) is deferred to v2.3.0 R7. No stub handler in v2.2.2 (Warden ruling).

### H3 — Inference thresholds consume contract
- `canonical/runtime-assets/shared/lib/deliverable-type-profile.mjs::inferDeliverableTypeFromWorkType` accepts optional 4th parameter `thresholdsConfig` and replaces previous integer magic numbers with a normalized ratio computation (`absoluteFactor * 0.6 + marginFactor * 0.4`) compared against contract-declared `confidenceThresholds`.
- **Backwards-compatible**: existing 3-arg callers continue to work with default thresholds `{ high: 0.85, medium: 0.6, low: 0.0 }`.
- Return shape gains `ratio` and `thresholds` fields for testability/audit.

## Added

- **`tests/poc-design-gate/05-rule-to-packet-mapper.test.mjs`** — 11 new tests covering H1 mapper (contract block presence, 7 core IDs, `resolvePacketName` mapped/fallback/invalid-input/null-registry, full-profile coverage) and H3 contract-threshold consumption (custom thresholds honored, defaults applied when omitted).
- Combined poc-design-gate suite: **59 tests** (48 v2.2.0 + 11 v2.2.2).

- **Meta-theory skill localization fix** — `canonical/skills/meta-theory/SKILL.md` adds the Chinese localization example `方案 A` next to the English `Option A` placeholder, and adds the Chinese fallback-card line `当前以聊天确认卡展示，不是弹窗。`. - All four runtime mirrors re-synced (`.claude/`, `.codex/`, `.cursor/`, `openclaw/`).
